# Bio-Synthesizer AI 🌿🧪

Bio-Synthesizer is an AI-powered tool that analyzes plants, identifies primary phytochemicals, and generates clinical scientific reports using **Local LLMs (Ollama)**.

## How to run?
1. Install [Ollama](https://ollama.com/) and run `ollama run llama3`.
2. Clone this repo: `git clone <your-repo-link>`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the app: `streamlit run app.py`

## Features
- Local AI Analysis
- PubChem Integration
- Human-in-the-loop Approval
